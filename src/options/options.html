LiveSub — Options

Translation

The hint is added to the prompt when using an AI provider, along with the video's title and recent lines — it helps a lot with genre-specific vocabulary. Machine translation providers ignore it.

“Google Translate (free)” uses an unofficial public endpoint — no key needed, but it is rate-limited and not guaranteed by Google. For heavy use, add a key for one of the official services below.

API keys

Keys are stored only in this browser profile (chrome.storage.local) and are sent only to the corresponding provider. They never touch any other server. Anyone with access to this browser profile could read them, so prefer keys with spending limits.

Custom OpenAI-compatible endpoint

Any service exposing /chat/completions (and optionally /audio/transcriptions): local Ollama/llama.cpp, OpenRouter, etc. Saving a custom URL will ask for permission to contact that host.

Live captions (speech-to-text)

Used when a video has no subtitles. Tab audio is recorded in short chunks and sent to an OpenAI-compatible /audio/transcriptions endpoint using your OpenAI key (or the custom key if no OpenAI key is set).
